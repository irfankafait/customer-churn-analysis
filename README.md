# Customer Churn Analysis

## Project Overview
This project analyzes customer churn patterns in a SaaS subscription business and builds a predictive model to estimate churn probability.

## Key Insights
- Customers with shorter tenure show higher churn risk
- Monthly contract customers churn more frequently
- High support ticket activity correlates with churn

## Model
Logistic Regression was used to estimate churn probability and identify high-risk customers.
